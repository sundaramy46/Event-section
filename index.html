OPEN SOURCE PROGRAMMING AND WEB DEVELOPMENT

Open source technology is the production and development philosophy of allowing end users and developers to not only see the source code of the software, but modify it as well. Any type of software program can be open source, including operating systems, databases, applications, games and even programming languages. In our workshop we taught them the concepts of Open Source programming and the basic GitHub commands and its complete uses, and to make open-source contributions. Along with this, we also taught them cloud computing which is the on-demand availability of computer system resources, especially data storage and computing power, without direct active management of the user. Our main focus for this workshop was on Amazon Web Services Console, we taught them various technology being used in web development with the main focus on JavaScript.
